# README

## 简介

github: https://github.com/SnoozeLee/webDemos

目前该仓库主要是用于存放一些前端小功能的代码, 不定时更新

目前主要包括两类代码

- demo : html/css/js 网页效果 (有的是绘画, 有的是动画, 有的是功能)
- exten : 浏览器插件
- sideTest : 测试语法、效果之类的, 帮助理解某些API或者css属性
